## Finalized Firestore Data Structures (Plan)

Based on discussion and requirements, the following structures will be used in Firestore:

**1. `users` Collection:**
   - **Document ID:** Firebase Authentication User UID (`user.uid`)
   - **Purpose:** Stores user profile information.

   ```json
   // Document ID: firebase_auth_user_uid
   {
     "email":           "user@example.com",     // String (From Auth)
     "displayName":     "User Name",            // String (From Auth or edited)
     "role":            "requester",            // String ("requester" or "deliverer")
     "address":         "Amanuensvagen 5",      // String
     "phone":           "+46 793570040",        // String
     "delivererStatus": "available",            // String | Null (e.g., "available", "unavailable", "on-delivery") - Primarily for deliverers
     "rating":          5,                      // Number | Null (Average rating for deliverers)
     "totalDeliveries": 10,                     // Number | Null (Completed deliveries count for deliverers)
     "createdAt":       Timestamp,              // Firestore Timestamp (Profile creation)
     "updatedAt":       Timestamp               // Firestore Timestamp (Last profile update)
     // uid field is the document ID itself, not stored inside the document.
   }
   ```

**2. `orders` Collection:**
   - **Document ID:** Auto-generated by Firestore
   - **Purpose:** Stores details for each grocery order.

   ```json
   // Document ID: auto_generated_firestore_id
   {
     // --- Requester Info (Denormalized) ---
     "requesterUid":     "user_auth_uid_string",     // String (User UID)
     "requesterName":    "User Display Name",        // String
     "requesterAddress": "Amanuensvagen 5",          // String
     "requesterPhone":   "+46 793570040",            // String

     // --- Order Details ---
     "items": [                                     // Array of Maps
       {
         "productId": "prod_123",                   // String
         "name":      "Milk",                       // String
         "price":     25,                           // Number (Price per unit at order time)
         "quantity":  1,                            // Number
         "imageUrl":  "https://.../milk.png"        // String (Optional)
       }
     ],
     "itemSubtotal":   85,                          // Number (Sum of item prices * quantity)
     "deliveryFee":    15,                          // Number
     "totalPrice":     100,                         // Number (itemSubtotal + deliveryFee)

     // --- Status & Workflow ---
     "status":         "Unassigned",                // String ("Unassigned", "Assigned", "PickedUp", "Delivering", "Delivered", "Cancelled")
     "paymentStatus":  "Pending",                   // String ("Pending", "PaidOnline", "CashOnDelivery", "Received")

     // --- Deliverer Info (Null until assigned) ---
     "delivererUid":   null,                        // String | Null (Deliverer User UID)
     "delivererName":  null,                        // String | Null
     "delivererPhone": null,                        // String | Null

     // --- Timestamps ---
     "createdAt":      Timestamp,                   // Timestamp (Order placed)
     "assignedAt":     null,                        // Timestamp | Null
     "pickedUpAt":     null,                        // Timestamp | Null
     "deliveredAt":    null,                        // Timestamp | Null
     "cancelledAt":    null                         // Timestamp | Null
   }
   ```
